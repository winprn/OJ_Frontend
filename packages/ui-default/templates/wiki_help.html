{{ _('Compile Error') }}

There may be the following situations:

  • 1. Submitting code but selected wrong language.
  • 2. Java main class name does not use "Main".
  • 3. For C/C++, check below.
  • 4. General compilation errors.

For C/C++ users：

  • 1. '__int64' should be written as 'long long' in GNU C++
  • 2. main() function should return 'int' instead of 'void'
  • 3. The variable i in the statement isn't a global variable, it'll become invalid after a new definement such as 'for(int i = 0...) {...}'.
  • 4. itoa() is not a general ANSI function （and there is no function in standard C/C++）
  • 5. %lf format in printf() isn't available and should be %f or %g

{{ _('Contest') }}

In OI, the last submission will be counted as the result.

In OI, highest total score is ranked first. Same score will be same rank.

In ACM/ICPC, passed code will be counted as the result.

In ACM/ICPC, the highest total number of problems solved is ranked first. If they're same, penalty will decide.

{{ _('Difficulty Algorithm') }}

The difficulty of a problem in {{ name }} is calculated base on number of submissions, pass rate and the submission running time and results of each submission.

Therefore, please note the following：

  • 0. The greater the difficulty value, the harder the problem is.
  • 1. Some easy problems difficulty may be inaccurate; The problem difficulty will become more accurate if and only if a large number of submissions has been made.
  • 2. The earlier the evaluation result of the user code submitted, the greater the impact on the difficulty of the topic.
  • 3. The difficulty also based on algorithm so there may be inaccurate results.
